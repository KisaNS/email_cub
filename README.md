# Email Cub

Простой веб-конструктор email-писем:

- drag-and-drop блоки (заголовок, текст, изображение, кнопка);
- настройка параметров выбранного блока;
- интерактивный предпросмотр письма (можно кликать по кнопкам/ссылкам);
- генерация итогового HTML-кода письма в табличной вёрстке;
- копирование сгенерированного кода.

## Запуск

Откройте `index.html` в браузере.
